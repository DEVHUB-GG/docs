# 🪪 LICENSE SYSTEM

{% hint style="info" %}
[Tebex Store](https://devhub.gg/product/7477007)
{% endhint %}

The DevHub License System lets you design license templates in a visual creator, issue them to
players, manage their status (active, expired, suspended, revoked, lost), and verify them at
scan terminals. It also ships an underground "fake ID" workflow with crafting stations and a
delivery mission.

## <mark style="color:yellow;">**Features**</mark>

* **Visual license creator** — design license templates with layouts, backgrounds, textures,
  components, icons, and per-field data binding.
* **License MDT** — dashboard, license lookup, profile search, permission management, and status
  changes.
* **Card holder & favorites** — players carry, show, give, take, and destroy their licenses.
* **License pickup NPC** — players purchase and pick up licenses from a city clerk.
* **Fake ID system** — underground NPC, crafting stations (paper cutter, foil tipper, embosser),
  and a fake-license delivery mission.
* **Scan terminals** — verify licenses with a customizable scan callback.
* **Physical or virtual licenses** — use inventory items or the built-in commands.
* **Laptop app integration** — optionally run the whole system as a `devhub_laptop` app.
* **Discord logging & open functions** — hook into every action server- and client-side.
