# 💻 Installation

{% stepper %}
{% step %}
### Install resources from keymaster

* Download the <mark style="color:red;">LOADINGSCREEN</mark> script file from keymaster.&#x20;
{% endstep %}

{% step %}
### Start resources

Move the files to the `resources` folder on your server and add the following lines to your server.cfg in the correct order:

```javascript
ensure devhub_loadingscreen
```
{% endstep %}

{% step %}
### Restart your server
{% endstep %}
{% endstepper %}
