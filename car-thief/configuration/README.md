# 🛠️ Configuration

Car Thief is configured through four files in the `configs` folder:

* [`sh.config.lua`](sh.config.lua.md) — the main gameplay configuration (locations, vehicles, car checker, key cloner, rewards, queue, integrations).
* [`sh.config_translations.lua`](sh.config_translations.lua.md) — every player-facing string (UI labels and notifications).
* [`client.lua`](client.lua.md) — the client-side police alarm hook.
* [`server.lua`](server.lua.md) — the server-side police alarm hook.
