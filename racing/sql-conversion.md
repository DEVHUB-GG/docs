# SQL Conversion

### <mark style="color:yellow;">Available Conversions</mark>

Currently, we support conversion for:

* **User Profiles** (Available Now)
* **Maps** (Planned for future update)

***

### <mark style="color:yellow;">Conversion Commands</mark>

Command can be used ONLY IN SERVER CONSOLE

***

#### <mark style="color:yellow;">User Profile Conversion</mark>

Converts user profiles from AV Racing script.

```
/racingConvert:av
```

Converts user profiles from RAHE Racing script.

```
/racingConvert:rahe
```

***

### <mark style="color:yellow;">Important Notes</mark>

* **Make a backup** - Always backup your database before running any conversion commands.

***

### <mark style="color:yellow;">Need Help?</mark>

If you're using a racing script that is **not included** in our conversion tools, please let us know!

We'll review your request and potentially add support for additional scripts in future updates.
